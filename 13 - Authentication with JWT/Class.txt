- npm init -y

- api deside karna kya kya aapi banana hai
- first server create in src/app.js and export to use in server.js
- connect with database by creating connection logic src/db/db.js and use it in app.js
- dotenv configuration using dotenv package for protecting credentials and create .env file also *env configuration must be in server.js file
- in .env file paste credeitials safely and in .gitignore file dont forget to use write node_modules and .env
